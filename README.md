# 🐍 Snake Game - 모바일 앱

클래식 Snake 게임을 현대적인 모바일 앱으로 재구현한 프로젝트입니다. React Native와 Expo를 기반으로 iOS, Android, 웹에서 모두 플레이할 수 있습니다.

**웹 브라우저에서 즉시 플레이:** [https://8081-i0kk4x5y8ohqakpridq5w-f5433698.sg1.manus.computer/](https://8081-i0kk4x5y8ohqakpridq5w-f5433698.sg1.manus.computer/)

## ✨ 주요 기능

### 게임 플레이

- **20x20 그리드 게임 보드** — 클래식 Snake 게임의 표준 크기
- **뱀 조종** — 스와이프 제스처 또는 D-pad 버튼으로 상/하/좌/우 조작
- **음식 먹기** — 음식을 먹을 때마다 뱀의 길이가 증가하고 점수 획득
- **충돌 감지** — 벽이나 자신의 몸과 충돌하면 게임 오버
- **난이도 선택** — 3단계 난이도로 게임 속도 조절
  - **쉬움** (Easy): 200ms 간격, 점수 1배
  - **보통** (Normal): 140ms 간격, 점수 2배
  - **어려움** (Hard): 90ms 간격, 점수 3배

### UI/UX 기능

- **점수 시스템** — 현재 점수와 최고 점수 실시간 표시
- **일시정지/재개** — 게임 중 언제든 일시정지 가능
- **게임 오버 화면** — 최종 점수와 신기록 여부 표시
- **최고 점수 저장** — AsyncStorage를 통한 로컬 저장
- **햅틱 피드백** — 버튼 클릭 및 게임 이벤트에 반응하는 진동 피드백

### 디자인

- **다크 네온 그린 테마** — 게임보이 스타일의 클래식하면서도 현대적인 디자인
- **반응형 레이아웃** — 모든 화면 크기에 최적화된 UI
- **모바일 우선 설계** — 한 손 조작 가능한 직관적인 인터페이스

## 🚀 시작하기

### 필수 요구사항

- Node.js 18.0 이상
- npm 또는 yarn 패키지 매니저
- Expo CLI (선택사항)

### 설치 및 실행

```bash
# 저장소 클론
git clone <repository-url>
cd snake-game

# 의존성 설치
npm install
# 또는
yarn install

# 개발 서버 시작
npm run dev
# 또는
yarn dev
```

### 플랫폼별 실행

**웹 브라우저:**
```bash
npm run dev
# 브라우저에서 http://localhost:8081 접속
```

**iOS (macOS에서만 가능):**
```bash
npm run ios
```

**Android:**
```bash
npm run android
```

**Expo Go 앱 (모바일):**
```bash
npm run qr
# 생성된 QR 코드를 Expo Go 앱으로 스캔
```

## 🔧 개발 가이드

### 게임 엔진 (`hooks/use-snake-game.ts`)

게임의 핵심 로직은 `useSnakeGame` 훅에 구현되어 있습니다:

- **상태 관리** — 뱀 위치, 음식 위치, 점수, 게임 상태
- **게임 루프** — 일정 간격으로 뱀을 이동하고 충돌 감지
- **점수 시스템** — 난이도별 점수 배수 적용
- **저장 기능** — AsyncStorage를 통한 최고 점수 저장

### 게임 보드 (`components/game-board.tsx`)

게임 보드 렌더링 컴포넌트:

- **그리드 라인** — 20x20 그리드 시각화
- **뱀 렌더링** — 머리와 몸통 구분하여 표시
- **음식 렌더링** — 동적 위치 업데이트

### 스타일링

프로젝트는 NativeWind (Tailwind CSS)를 사용합니다:

// 예시

  </Text>

## 📊 성능 최적화

- **메모이제이션** — `React.memo`로 불필요한 리렌더링 방지
- **Ref 활용** — 게임 루프에서 최신 상태 참조
- **효율적인 충돌 감지** — Set 자료구조 활용
- **화면 유지** — `useKeepAwake` 훅으로 게임 중 화면 꺼짐 방지

## 🐛 알려진 문제 및 해결책

| 문제 | 원인 | 해결책 |
|------|------|--------|
| 게임이 느려짐 | 많은 뱀 세그먼트 렌더링 | 난이도를 낮추거나 게임 재시작 |
| 터치 반응 없음 | 제스처 감지 실패 | 앱 재시작 또는 브라우저 새로고침 |
| 최고 점수 미저장 | AsyncStorage 오류 | 앱 캐시 삭제 후 재시작 |
